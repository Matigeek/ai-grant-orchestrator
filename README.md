# AI Grant Orchestrator

🚧 Early-stage open-source project | Actively developed

AI-powered system for matching researchers with international funding opportunities and building personalized research career paths.

---

## 🚀 Overview

AI Grant Orchestrator is an open-source initiative designed to transform how researchers discover funding opportunities and plan their scientific careers.

The system enables:

- intelligent matching between researchers and international funding calls (Horizon Europe, EIT Health, Erasmus+)
- personalized research career pathway design
- automated discovery, filtering, and analysis of grant opportunities
- AI-assisted positioning of research ideas and proposals

The goal is to shift research support from a reactive model to a proactive, data-driven and AI-enhanced approach.

---

## 🎯 Target Users

- Researchers and PhD candidates  
- Technology Transfer Offices (TTOs)  
- Innovation and grant support departments  
- Universities and research institutions  
- Early-stage MedTech and Digital Health startups  

---

## 🧠 Key Features

- 🔍 **AI Grant Matching**  
  Matches researcher profiles with relevant funding opportunities across EU and international programs  

- 🧭 **Career Path Generation**  
  Builds personalized, multi-year development pathways aligned with funding ecosystems  

- 📊 **Call Analysis Engine**  
  Extracts key requirements, success factors, and positioning strategies from funding calls  

- ⚙️ **Automation Workflows (n8n)**  
  Enables scalable, repeatable grant discovery and evaluation pipelines  

- 🧩 **RAG-based Knowledge System**  
  Integrates structured knowledge from funding databases and institutional expertise  

---

## 🏗️ Architecture (Concept)

- LLM-based reasoning layer (Claude / OpenAI)  
- Retrieval-Augmented Generation (RAG)  
- Workflow automation (n8n)  
- Researcher profile input layer  
- Funding database integration (planned)  

---

## 🔧 Tech Stack

- n8n (workflow orchestration)  
- Python / Node.js (planned)  
- LLM APIs (Claude / OpenAI)  
- VS Code + Copilot  
- Vector database (RAG layer)
